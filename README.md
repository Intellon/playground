# Playground

A collection of standalone Java programs, code snippets, and tutorials showcasing various concepts and techniques — from small utility classes to mini-games and educational walkthroughs.

## Code Snippets (`codesnippets`)

- **ChromeProcessHandlerWindows** — Handles Chrome processes on Windows (kill, restart, monitor).
- **ChromeProcessScannerWindows** — Scans for running Chrome processes on Windows.
- **CosineSimilarity** — Computes cosine similarity between two strings using word-frequency vectors.
- **CrossBrowserProcessHandlerSimple** — Simple cross-browser process handling example.
- **MailClient** — Demonstrates sending email via Jakarta Mail.
- **MavenEnvHandler** — Reads and manages Maven environment configurations.
- **RestHandler** — Basic RESTful service handler example.

## Games (`games`)

- **brickbreaker** — Classic Brick Breaker arcade game built with Java Swing.
- **minesweeper** — Minesweeper game with graphical UI and mine-sweeping logic.
- **panzergame** — Tank battle game with sprite-based rendering.

## Tutorials (`tutorials`)

- **base64converter** — Standalone application to encode and decode strings using Base64.
- **calculator** — Simple calculator demonstrating basic arithmetic operations.
- **database** — Utilities and helpers for database operations.
- **filelauncher** — Launches external files and applications from Java.
- **greeter** — Command-line greeting program showcasing user interaction.
- **loadgeneratoripconfigurator** — Tools for load generation and IP configuration testing.
- **processhandling** — Demonstrates OS-level process management.
- **shadowdom** — Selenium/Selenide utilities for reading and interacting with Shadow DOM elements, including support for open and closed shadow roots.
- **StartBinaryBanner** — Displays banners in binary format.
- **StartInterestsRates** — Simple interest rate calculator.
- **StartSquares** — Calculates and displays squares of numbers.
- **StartWhatsYourNumber** — Interactive number-guessing game.

## Usage

Each program or code snippet can be run independently. Navigate to the respective package and compile the Java files as per standard Java compilation procedures, or run them directly from your IDE.
